Divide and Conquer


"Divide and Conquer" is an online application, whose development I contributed to as a part of the jovially named team (ET)=>{ phone("home")}. The app is a turn-based multiplayer board game in which the player must attempt to take control of as much of the board as possible. Divide and Conquer is equal parts strategy and chance, and the player with the highest score at the end of 15 turns will emerge the victor.

The app leverages google's ouath API to verify users and associate each game's end result with the two individuals who played it. This user data persistance is what powers the app's leaderboard shown on the home page. The websocket library socket.io, in conjunction with Redis Cache, power the main functionalities of the game itself, with Redis being particularly critical in creating and managing the app's "lobby" where users await a match.

TackIt


"TackIt" is a web-based chat application that I helped develop for a class project. The idea behind TackIt was to create a live chat application with devlopment aides (Trello, GitHub, etc.) integrated for ease of use and improved communication. While we made significant progress towards achieving that goal, we were limited by our one week development timeframe. Furthermore, this chat app is almost entirely static, and only relies on Google's firebase for data persistance.

TackIt uses google oauth for user authentication, as well as firebase to power the chat's persistance / event driven update routine across all users. With a rudimentary "channel" system and contacts list, it was an accomplishment given the limitiations on time and tech available for the assignment.

Bamazon


"Bamazon" is a command-line shopping application written in node. The app, while not actually connected to a central server or management system, attempts to emulate what that would look and feel like. It stores product and user data in a MySQL database, which it uses to dynamically generate the on-page content. The app also leverages the node package inquirer to create an intuitive CLI experience. Bamazon not only allows the user to select products from his/her "cart", but also features administrative tools to update product inventory levels, adjust pricing, approve new administrator accounts, and more.
